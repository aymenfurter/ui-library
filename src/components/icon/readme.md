# Icon

The Baloise SVG icons have been converted with this [tool](https://websemantics.uk/tools/svg-to-background-image-conversion/).

## Usage

Here is how you can change the size:

```html
<div class="columns">
  <div class="column">
    Font
  </div>
  <div class="column">
    SVG
  </div>
</div>
<div class="columns">
  <div class="column">
    <bal-icon name="date" is-font="true" size="small"></bal-icon>
  </div>
  <div class="column">
    <bal-icon name="date" size="small"></bal-icon>
  </div>
</div>
<div class="columns">
  <div class="column">
    <bal-icon name="date" is-font="true"></bal-icon>
  </div>
  <div class="column">
    <bal-icon name="date"></bal-icon>
  </div>
</div>
<div class="columns">
  <div class="column">
    <bal-icon name="date" is-font="true" size="medium"></bal-icon>
  </div>
  <div class="column">
    <bal-icon name="date" size="medium"></bal-icon>
  </div>
</div>
<div class="columns">
  <div class="column">
    <bal-icon name="date" is-font="true" size="large"></bal-icon>
  </div>
  <div class="column">
    <bal-icon name="date" size="large"></bal-icon>
  </div>
</div>

<bal-icon name="account" size="small"></bal-icon>
<bal-icon name="account"></bal-icon>
<bal-icon name="account" size="medium"></bal-icon>
<bal-icon name="account" size="large"></bal-icon>
```
